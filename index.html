HPS 281 SoCal

This is the Sierra Club's "Hundred Peaks Section" for Southern California.

The full list has the tallest 281 peaks they recommend climbing in the area.

This API will serve you the 6 tallest peaks. Please use this URL:
https://hps-281-socal-api-c3f4c9431c63.herokuapp.com/api/

Add the following peaks to the end of the URL to get their data:

  • gorgonio
  • jepson
  • bighorn
  • dragonshead
  • anderson
  • jacinto

You can also enter one of the 6 names above into the input field below and click the search button to display the data on this page.
